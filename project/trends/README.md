# Daily Trend Reports

This directory contains daily trend reports generated by the Trend Finder module.

## Structure

```
project/trends/
├── index.json              ← master index of every daily report
├── 2024-01-15.json         ← daily report (top 10 trends)
├── 2024-01-16.json
└── README.md
```

## Report JSON format

```json
{
  "date": "2024-01-15",
  "trends": [
    {
      "rank": 1,
      "topic": "AI tool that replaced my entire team",
      "keywords": ["ai tools", "automation", "chatgpt", "productivity"],
      "niche": "ai_tools",
      "viralScore": 87.5,
      "sources": ["youtube_trending", "reddit_r/artificial_hot", "google_trends_daily"]
    }
  ],
  "_meta": {
    "generatedAt": "2024-01-15T09:00:00.000Z",
    "totalRawTrends": 312,
    "sourceCounts": {
      "youtube_trending": 40,
      "youtube_search": 80,
      "google_trends_daily": 20,
      "google_trends_interest": 25,
      "reddit_r/artificial_hot": 25
    },
    "topicCount": 10
  }
}
```

## Viral Score

The `viralScore` (0–100) is a weighted composite of:

| Signal | Weight |
|--------|--------|
| YouTube trending position | 40% |
| YouTube search view velocity | 35% |
| Google Trends daily ranking | 15% |
| Google Trends interest score | 10% |
| Reddit upvote ratio + recency | 8–15% |

Bonuses applied:
- **+15** per additional source confirming the topic (max +30)
- **+10** if topic matches a target niche
- **+5** if 4+ unique keywords extracted

## Niches

| Niche | Description |
|-------|-------------|
| `ai_tools` | AI tools, ChatGPT, automation apps |
| `tech_facts` | Surprising technology facts |
| `automation` | Workflow automation, no-code tools |
| `money_facts` | Personal finance, investing, wealth |
| `productivity` | Habits, time management, performance |
